# 函数展示模板


## 页面标题

`函数名: 一句话说明这个函数解决什么问题`

## 1. 适用场景

- 这个函数解决什么问题
- 什么时候应该用它
- 和哪些函数常一起使用

## 2. 输入与输出

| 项目 | 说明 |
|---|---|
| 输入 | `xarray.DataArray` / `Dataset` / 文件路径 / 关键参数 |
| 输出 | 返回对象类型、维度结构、是否可直接绘图或保存 |
| 前置条件 | 时间长度、纬向范围、是否需要标准化 |

## 3. 最小可运行示例

```python
from tropical_wave_tools import open_example_olr
from tropical_wave_tools.<module> import <function_name>

data = open_example_olr()
result = <function_name>(data)
```

## 4. 典型结果

- 放一张最能说明函数价值的结果图
- 图下说明结果应该如何解读

## 5. 参数建议

- 默认参数何时够用
- 哪些参数最常需要调整
- 参数变化会对结果产生什么影响

## 6. 与其他函数的关系

- 上游函数：数据读取、预处理
- 下游函数：绘图、统计、保存、对比

## 7. 注意事项

- 数据维度命名
- 时间长度要求
- 滤波或频谱方法的适用范围
- 性能与内存注意点
