# M05 EOF 模态与 PC 回归

## 对应代码

- `src/tropical_wave_tools/eof.py`
- `src/tropical_wave_tools/atlas.py`
- `src/tropical_wave_tools/plotting.py`

## 解决的问题

EOF 用于从滤波后的波动场中提取主导空间模态，PC 回归用于把 OLR 主模态与风场响应联系起来。它回答的问题是：某个波段最主要的空间结构是什么？对应的风场异常如何配合？

## 数学原理

把异常场展平成矩阵：

```text
X(feature, time)
```

对它做奇异值分解：

```text
X = U * S * V^T
```

其中：

- `U` 对应 EOF 空间型。
- `S` 对应奇异值。
- `V` 对应 PC 时间序列。
- 方差贡献率由奇异值平方决定。

将其他变量回归到标准化 PC 上：

```text
beta(y, x) = cov(Y(t, y, x), PC(t)) / var(PC(t))
```

如果 PC 已标准化，`beta` 表示 PC 增强一个标准差时，变量 `Y` 的典型异常幅度。

## 物理意义

- EOF1/EOF2 常常可以表示传播波的两个相位。
- PC 时间序列描述该模态随时间的强弱变化。
- OLR EOF 与 U850/V850 回归图能显示对流和低层风场的耦合结构。
- EOF 正负号本身没有绝对意义，解释时应关注相对相位和空间结构。

## 实现要点

当前 atlas 的 EOF 流程会：

1. 对不同波型选择不同纬度范围。
2. 对 Kelvin、ER 等偏对称波使用对称投影。
3. 对 MRG 等偏反对称波使用反对称投影。
4. 使用 `sqrt(cos(lat))` 做面积权重。
5. 对滤波 OLR 做 SVD EOF。
6. 把 OLR、U850、V850 回归到 OLR PC 上。

## 近期实现修正

`n_harmonics=0` 现在表示只去掉时间平均，不再构造日气候态低阶谐波。这更适合已经完成波段滤波的异常场，避免在 EOF 前重复去季节循环。

## 注意事项

- EOF 模态可能发生符号翻转，这是数学等价的结果。
- EOF1/EOF2 是否组成传播相位对，需要结合 PC 相位、Hovmoller 和空间图判断。
- 如果样本长度短，EOF 方差贡献率不稳定。
- 对非 OLR 变量做 EOF 时，事件解释和物理含义要重新定义。
